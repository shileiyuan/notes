**坐标分两种： 文档坐标和视口坐标 (document vs viewport)**



### 查询窗口滚动条的位置

`window.scrollX`  `window.scrollY` 是 `window.pageXOffset` 和 `window.pageYOffset` 的别名



### window.scroll

```javascript
// 使用的坐标均为文档坐标
window.scrollTo(0, 1000);
// 或者使用 options
window.scrollTo({
  top: 100,
  left: 100,
  behavior: 'smooth' // smooth auto
});
```
