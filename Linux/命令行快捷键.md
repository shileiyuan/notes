## 光标移动

| 组合键   | 作用                                      |
| -------- | ----------------------------------------- |
| ctrl + A | 移动光标到行首                            |
| ctrl + E | 移动光标到行尾                            |
| ctrl + L | 清屏并把光标移动到左上角；与clear命令相同 |
| ctrl + F | 光标向前移动一个字符；和右箭头作用一样    |
| ctrl + B | 光标向后移动一个字符；和左箭头作用一样    |



## 修改文本

| 组合键   | 作用             |
| -------- | ---------------- |
| ctrl + D | 删除光标处的文本 |



### 剪切和粘贴命令

| 组合键   | 作用                         |
| -------- | ---------------------------- |
| ctrl + K | 剪切从光标到行尾的文本       |
| ctrl + U | 剪切从光标到行首的文本       |
| ctrl + Y | 把缓冲区的文本粘贴到光标位置 |
